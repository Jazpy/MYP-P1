Math function grapher.

Please use infix notation, for example: (x+5)/2

Supported functions and operators:
	unary '-'
	binary '-'
	binary '+'
	'*'
	'/'
	'^'
	sin()
	cos()
	tan()
	cot()
	sec()
	csc()
	sqr()

In order to use this application, please type in "./configure" while in the topmost directory,
followed by "make" and finally "src/main" to run the application.

If you want to run the unit tests, type in "make check" after having run "configure".
